## 在 Sealos 部署步骤

**1. 配置 AskTable**

`LLM_API_KEY` 是访问AI 模型的 API_KEY， 建议[登录 AskTable](https://cloud.asktable.com) ，右上角点击【免费下载 AskTable】免费申请。其他配置项可使用默认值。
